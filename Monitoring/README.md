# Monitoring

Notes and hands-on exercises for Monitoring.

## Hands-on project

[Production-Monitoring-Stack/](Production-Monitoring-Stack/) — a complete, production-grade
observability stack on Kubernetes (Grafana Alloy + Loki + Grafana + Prometheus), built from a
real-world draft configuration with its HA gaps and misconfigurations analyzed and fixed.
Good for practicing: Helm-based stack deployment, spotting "looks-HA-but-isn't" traps, secrets
management, and cross-component consistency (labels, storage, alerting).
