# Project 2: Docker + Kubernetes App

This project is for intermediate DevOps practice.

## Focus areas
- Docker image creation
- Kubernetes deployment manifests
- Rolling updates and rollback strategy
- Health checks and services

## Suggested steps
1. Containerize the app.
2. Create a Kubernetes Deployment and Service.
3. Add readiness and liveness probes.
4. Practice a rolling update and rollback.

## Deliverables
- Dockerfile
- Kubernetes deployment.yaml
- service.yaml
- README with rollout explanation
