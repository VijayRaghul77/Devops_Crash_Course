# Project 1: Basic CI/CD App

This is a beginner-friendly DevOps project to practice:
- Git workflow
- CI pipeline basics
- Docker build and run
- Simple deployment validation

## Goal
Build a small web app, add tests, and automate a simple CI pipeline.

## Suggested steps
1. Clone the repo and work on this project folder.
2. Add a simple test script.
3. Add a GitHub Actions workflow.
4. Build and run the container locally.
5. Explain the pipeline end to end in interviews.

## Deliverables
- App source code
- Test script
- Dockerfile
- CI workflow
- README with run instructions
