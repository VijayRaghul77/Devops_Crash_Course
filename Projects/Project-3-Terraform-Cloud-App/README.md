# Project 3: Terraform + Cloud App

This project is for advanced DevOps interview practice.

## Focus areas
- Terraform basics
- Cloud provisioning
- Remote state
- Environment separation

## Suggested steps
1. Create Terraform files for a simple resource.
2. Initialize Terraform and review the plan.
3. Apply the configuration.
4. Explain how infrastructure changes are versioned and reviewed.

## Deliverables
- main.tf
- variables.tf
- outputs.tf
- README with architecture explanation
