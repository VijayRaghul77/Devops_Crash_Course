# DevOps Interview Preparation Roadmap

## Repo analysis
The repository is currently a blank starting point, which makes it suitable for building a personal DevOps interview lab.

## Recommended topic order
1. Linux fundamentals
2. Git and GitHub workflows
3. Docker and containers
4. CI/CD pipelines
5. Networking basics
6. Kubernetes basics
7. Terraform and IaC
8. Monitoring and troubleshooting
9. Cloud basics (AWS/Azure)

## Hands-on experiments
- Docker lab: build, run, test a containerized app
- Git + CI lab: practice branching, commits, and validation workflow
- Future labs to add:
  - Kubernetes deployment and service exposure
  - Terraform provisioning of a simple resource
  - Monitoring with logs and metrics

## Interview mindset
- Always explain what you are doing before running commands
- Practice troubleshooting with logs and error messages
- Be ready to answer why, not just how
- Focus on real scenarios rather than memorization
